# Zero Assistant

Zero Assistant is a virtual assistant program built in Python. It can perform various tasks based on voice or text commands provided by the user.

## Features

- Voice and text command support: Users can interact with the assistant using either voice or text commands.
- Web search: Search the web for information on any topic.
- YouTube video playback: Play YouTube videos based on user requests.
- Time and date: Get the current time and date.
- Virus scan: Run a virus scan using the Windows built-in tool.
- Wikipedia search: Get information on any topic from Wikipedia.
- Calculator: Open a simple calculator for basic arithmetic operations.
- Notepad: Open the Windows Notepad application.
- Android Studio: Open Android Studio for Android app development.
- Discord: Open the Discord messaging application.
- Eyes and virtual mouse control: Control the mouse cursor using hand gestures or facial movements.
- Internet speed test: Check the download and upload speed of the internet connection.
- Battery status: Check the current battery percentage and status.
- Jokes: Get a random joke for some humor.

## Getting Started

To use Zero Assistant, follow these steps:

1. Install the required Python libraries using pip:

```
pip install tkinter opencv-python-headless mediapipe pyautogui psutil speedtest-cli pywhatkit wikipedia
```

2. Clone or download the Zero Assistant repository to your local machine.

3. Run the `zero_assistant.py` file to start the assistant.

## Usage

Once the assistant is running, you can interact with it using either voice or text commands. Some example commands include:

- "Search for [query]": Perform a web search for the specified query.
- "Play [song or video]": Play a song or video from YouTube.
- "What is the time?": Get the current time.
- "Scan for viruses": Run a virus scan on the system.
- "Tell me about [topic]": Get information about a specific topic from Wikipedia.
- "Open calculator": Launch the calculator application.
- "Open Notepad": Open the Windows Notepad application.
- "Open Android Studio": Launch Android Studio for app development.
- "Open Discord": Open the Discord messaging application.
- "Control mouse with eyes": Control the mouse cursor using eye movements.
- "Test internet speed": Check the download and upload speed of the internet connection.
- "Check battery status": Get information about the current battery status.
- "Tell me a joke": Get a random joke for some humor.
